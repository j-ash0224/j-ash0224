- 데이터베이스 설계 및 생성: 프로젝트 요구사항에 맞는 데이터베이스 스키마 설계 및 생성

- EC2: 가상 서버를 생성하여 웹 서버 및 데이터베이스 서버 호스팅

- Cloud Architecture 설계: Serverless, Microservice Architecture 등 다양한 아키텍처 패턴을 이해하고 설계
